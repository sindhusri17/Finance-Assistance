# AI-Powered Personal Finance Assistant

## Project Description
A Flask-based web application that helps users track expenses, manage budgets, monitor financial health, and receive savings recommendations.

## Features
- Secure Login
- Expense Tracking
- Monthly Budget Management
- Savings Recommendation
- Financial Health Analysis
- Transaction History
 
## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
